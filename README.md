# Krovat-Tournament.Tournament-System

(BG)
Логическо ядро на туртирна система без графичен интерфейс. Турнирния формат е Double Elimination - поток на победителите и губещите.

1. Описание
Приема списък от произволен брой отбори от конзолата или файл. При въвеждане на информация за мачовете симулира придвижването на победителя. В края на турнира извежда победителя и историята на мачовете.
